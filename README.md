# 🎬 FastAPI Movie Ticket Booking System

## 📌 Project Description
This project is a backend system built using FastAPI for managing movie bookings.

## 🚀 Features
- GET APIs
- POST APIs with Pydantic validation
- CRUD operations
- Multi-step workflow (seat hold → confirm → release)
- Search, Sorting, Pagination
- Swagger API testing

## 🛠 Tech Stack
- Python
- FastAPI
- Uvicorn

## ▶️ Run Project
```bash
uvicorn main:app --reload
